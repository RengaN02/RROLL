# ROLL

**ROLL** is an indie hobby game project featuring an isometric (angled top-down) perspective. It doesn't have many mechanics right now, but I plan to add more features over time.

## Visual Style

The defining visual mechanic of the game is a **light-sensitive dither effect** applied over everything.

## About the Project

This game is developed entirely as a personal **hobby project** using **Unity**. The main goal is to experiment with game mechanics, explore visual techniques (such as Shader Graph), and learn throughout the development process.
